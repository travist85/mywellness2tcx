How to Export Workouts from Technogym MyWellness

This guide explains how to get your workout data out of MyWellness so you can convert it to TCX or FIT. It is written for users with no technical background.

Outcome: you will have a ZIP export that can be uploaded directly into the converter.

What you can export

In MyWellness, machine sessions are typically stored in files named indooractivities-*.json inside your export ZIP. The converter reads these files directly.

Export steps

  1. Sign in to your MyWellness account on desktop.
  2. Open account settings and look for data/privacy export options.
  3. Request and download your archive when it is ready.
  4. Save the ZIP file without changing its contents.
  5. Open the converter and use Upload ZIP.

Quick checks before upload

  • Do not extract and re-zip unless necessary.
  • Make sure the ZIP contains at least one file named indooractivities-*.json.
  • If your export is old, request a fresh one and retry.

If ZIP export is missing or incomplete

Use the converter's Paste JSON mode for a single workout. You can copy a workout response from your browser dev tools. If needed, follow this troubleshooting guide: Fix JSON import issues.
