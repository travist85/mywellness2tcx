How to Import TCX/FIT into Garmin and Strava

After converting a Technogym/MyWellness workout, choose where you want to upload the file. FIT usually has better compatibility, but TCX is still useful in many tools.

If one format fails, try the other before troubleshooting deeper.

Garmin Connect

  1. Open Garmin Connect on the web.
  2. Go to Import Data (or the upload option in Activities).
  3. Select your .fit or .tcx file.
  4. Upload and review the activity details.

Strava

  1. Open Strava on the web.
  2. Use Upload Activity then choose File.
  3. Select your converted file and complete upload.
  4. Edit title/sport type if needed.

Which format should you use?

  • FIT: Best default for compatibility and richer activity fields.
  • TCX: Good fallback for tools that specifically expect TCX XML.

Common import issues

  • Rejected file: Re-export with FIT and retry.
  • No distance shown: Ensure distance export toggle was enabled before download.
  • Duplicate activity: Delete the earlier upload and keep one copy.
  • Wrong sport type: Edit activity type in the destination app after upload.
